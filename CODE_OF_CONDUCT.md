# Code of Conduct

## Our Standard

This project expects respectful, direct, and technically grounded collaboration.

Examples of behavior that contribute to a healthy environment:

- giving actionable technical feedback
- disagreeing on design choices without making it personal
- documenting assumptions and tradeoffs clearly
- respecting privacy and local-data boundaries

Examples of unacceptable behavior:

- harassment or personal attacks
- dismissive or hostile review behavior
- publishing sensitive user data, paths, or local content without permission
- bad-faith disruption of issues, discussions, or pull requests

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that violate this code of conduct.

## Reporting

If you encounter a serious conduct problem, open a private maintainer contact path before escalating publicly.
